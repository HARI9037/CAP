# CAP — Context-Aware Partner
A contextual workflow AI agent with human-in-the-loop orchestration.
Built for the OpenAI × Outskill AI Builders Hackathon.

Core philosophy: THINK IN CLOUD → ACT LOCALLY → CONFIRM EVERYTHING

Stack:
- Frontend: React + Vite + TailwindCSS → Netlify
- Backend: FastAPI + Python 3.11 → Render
- Brain: Groq API
- Memory: SQLite
- Tools: httpx + BeautifulSoup, Workspace File Manager
- Confirmation: Human-in-the-loop gate before every action

## MVP Goal
Build ONE polished workflow continuation experience.
Prioritize reliability, clarity, and deployment stability over feature quantity.
